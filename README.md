# 🏨 Hotel Samurai

Sistema de reservas de hotéis e pousadas desenvolvido com HTML, CSS e JavaScript puro.

🔗 **Demo ao vivo:** [reservas-de-hoteis-xi.vercel.app](https://reservas-de-hoteis-xi.vercel.app)

## 📋 Sobre o Projeto

O Hotel Samurai é uma plataforma de busca e reserva de acomodações que permite aos usuários pesquisar hotéis e pousadas por localização, datas e período de estadia, além de verificar disponibilidade em tempo real.

## ✨ Funcionalidades

- 🔍 Busca de hotéis por localização, data de entrada/saída e período
- 🏠 Listagem de hotéis e pousadas em destaque
- 💰 Seção de promoções com preços especiais
- 📅 Verificação de disponibilidade de quartos
- 🔐 Sistema de login de usuários
- 📱 Layout responsivo para mobile e desktop

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (múltiplos arquivos modulares)
- JavaScript
- Bootstrap 5
- Font Awesome 4.7
- Google Fonts (Material Icons)
- jQuery / Popper.js
